# AI Client Acquisition Platform - Frontend

[cite_start]This repository contains the user interface and administrative dashboard for the AI Client Acquisition Platform. [cite_start]Built using React/Next.js, it provides marketing agencies with a clean, centralized interface to orchestrate outreach campaigns, monitor lead pipelines, review metrics, and manage human-in-the-loop approvals[cite: 8, 74, 201, 435].

### Key Features:
- **Campaign Management Panel:** Group, organize, filter, and track performance metrics across specific target niches and locations.
- **Advanced CRM Pipeline Dashboard:** Visual tracking of prospects through deep lifecycle stages (Lead → Contacted → Replied → Qualified → Meeting Scheduled → Won/Lost).
- [cite_start]**Human Approval Gate UI:** An administrative review interface allowing managers to preview, tweak, and batch-approve AI-drafted messages before they are dispatched[cite: 433, 450, 456].
- [cite_start]**Real-Time Analytics View:** Visual tracking of delivery rates, open rates, AI response performance, and total booked meetings[cite: 211].

### Tech Stack:
- [cite_start]React.js / Next.js Framework 
- Responsive CSS / Component Libraries (Tailwind CSS)
- Axios (API integration with the Django REST Backend)
